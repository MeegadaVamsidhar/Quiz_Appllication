# Quiz App

## Features

- Multiple-choice questions with instant feedback.
- Score tracking and high score display.
- Randomized question order for a fresh experience each time.
- Responsive design for mobile and desktop devices.

## Technologies Used

- HTML
- CSS
- JavaScript
